## 1.如何使用？
> 项目以库的形式使用，Qt子工程使用．再在其他子工程包含pri文件即可使用。

## 2.如何启用demo测试？
> 在QtNetworkService.pro文件中将"#CONFIG += QT_APP_MODE"这一行的#去除即可转为可执行文件形式，在Demo目录的main.cpp为主执行文件，如需要测试接口编辑便可。
